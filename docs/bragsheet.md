# Bragsheet

High-impact entries, organized by P&G cycle. Each row is produced by the `bragsheet-entry` skill or `bin/growth-capture`.

**Add an entry:** `pbpaste | bin/growth-capture`
**Roll up weekly signals:** `bin/bragsheet-rollup --append-file docs/bragsheet.md`
**Draft a P&G sync:** `bin/pg-sync --output-file docs/growth/cycles/YYYY-QN.md`
